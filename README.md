## 🤔 진행 방법
1. 프로젝트를 자신의 계정으로 fork
2. 기능 구현 후, commit 규칙에 따라 commit
3. 본인 원격 저장소에 올리기
4. 원본 레포지토리로 Pull Request

5. PR에 코드 리뷰
